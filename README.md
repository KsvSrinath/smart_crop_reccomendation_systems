# Smart Crop Recommendation System

This project recommends the most suitable crop based on soil and environmental conditions.

## Dataset Used
- Crop Recommendation Dataset from Kaggle

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## Features
- Input parameters: Nitrogen, Phosphorus, Potassium, Temperature, Humidity, pH, and Rainfall.
- Outputs the most suitable crop using a trained ML model (Random Forest Classifier).

## How to Run
1. Open `smart_crop_recommendation.ipynb` in Jupyter Notebook or VS Code.
2. Run all cells to see predictions.
